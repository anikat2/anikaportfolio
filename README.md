# Portfolio

A modern developer portfolio with animated particle hero, glassmorphism UI, and full sections: About, Skills, Experience, Projects, Blog, and Contact.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Customize

Edit copy, links, and stats in `src/data/content.ts`. Replace the placeholder avatar initials in `About.tsx` with your photo when ready.

## Build

```bash
npm run build
npm run preview
```
